## 食用方法

### 安装依赖

#### Linux

`pip3 install -r requirements.txt`

#### Windows

`pip install -r requirements.txt`

### 运行

#### Linux

`python3 app.py --path <path> --port <port>`

#### Windows

`python app.py --path <path> --port <port>`
